Für **GedankenRein Coaching** entstand eine **SEO‑starke WordPress‑Website** mit **responsivem Design** und klarer Informationsarchitektur. 

Die Seite nutzt semantisches HTML, sprechende Metadaten und strukturierte Daten (Schema.org), um die Sichtbarkeit in Suchmaschinen zu erhöhen. Performance‑Optimierungen wie **Lazy‑Loading**, optimierte Bildgrößen und minimierte Styles/Assets sorgen für schnelle Ladezeiten auf allen Geräten. Die Architektur ist auf künftige Features wie **Online‑Buchungen** vorbereitet.

### Sprachen & Tools
**Sprachen:** HTML5, CSS3, JavaScript  
**Frameworks/Tools:** WordPress, Bootstrap, jQuery, Lighthouse, Schema.org

> **Ergebnis:** Eine schnelle, gut auffindbare Website mit einfacher Pflege im CMS und klarer Erweiterbarkeit.
