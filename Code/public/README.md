# r³webdesign – High-Performance Webentwicklung & UI/UX

Willkommen im Repository von **r³webdesign**. Nach der erfolgreichen Migration auf **Astro** dient diese Plattform als Aushängeschild für moderne, skalierbare und blitzschnelle Web-Lösungen. Hier trifft technisches Engineering auf präzises Design.

## 🚀 Der Astro-Vorteil (Migration 2026)
Das Projekt wurde von einer klassischen Struktur auf **Astro** umgestellt, um die Vorteile der *Island Architecture* und *Static Site Generation (SSG)* voll auszuschöpfen.

* **Performance:** Nahezu 100/100 Lighthouse-Scores durch minimales Client-Side JavaScript.
* **SEO:** Automatisierte Sitemap-Generierung (`sitemap-index.xml`) und strukturierte Metadaten.
* **Wartbarkeit:** Modulare Komponenten und Content Collections für Projekte.

## ✨ High-End Features
* **Lando-Norris-Vibe:** Interaktive Hintergrund-Animationen mit SVG-Turbulenzen, Backdrop-Blurs und organischen Wellen-Effekten.
* **Ambient Spot Emitter:** Ein dynamisches System aus im Hintergrund driftenden Lichtpunkten ("Orbs") für maximale visuelle Tiefe.
* **Responsive Code Window:** Ein intelligentes Terminal-Fenster im Tech-Stack, das Code-Beispiele gerätespezifisch optimiert darstellt.
* **Interaktives Grid:** Ein technisches HUD-Overlay, das auf Mausbewegungen reagiert und die Professionalität der UI unterstreicht.

## 🛠 Tech Stack
* **Framework:** [Astro](https://astro.build/)
* **Sprachen:** HTML5, Modern CSS (SCSS-Logik), JavaScript (ES6+)
* **Tooling:** Vite, PostCSS, ESLint
* **Deployment:** Automatisierter CI/CD Workflow über GitHub Actions
* **Hosting:** Optimiert für r3webdesign.de via Vercel / Netlify

## 📁 Projektstruktur
* `src/pages/`: Datei-basiertes Routing für alle Unterseiten.
* `src/layouts/`: Zentrales `BaseLayout.astro` für konsistentes Branding.
* `src/styles/`: Modulare Stylesheets (z.B. `subtle-bubbles.css`) und globale Variablen.
* `public/`: Statische Assets wie `robots.txt`, Favicons und Projektbilder.
